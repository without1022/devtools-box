# 开发工具箱 — 开发计划

## 目标

做一个**好用的开发者工具站**：纯前端、无需后端、单文件部署、工具全面、体验干净。
对标 it-tools.tech，面向中文开发者优化。

## 开发队列

### 阶段一：补全开发者刚需（P0）

| # | 工具 | 说明 |
|---|---|---|
| 19 | Cron 表达式 | 输入 cron 字符串 → 人话描述 + 最近5次执行时间 |
| 20 | 进制转换 | 2/8/10/16 进制互转，支持 2-36 任意进制 |
| 21 | 命名风格转换 | camelCase ↔ snake_case ↔ PascalCase ↔ kebab-case |
| 22 | 文件 Hash | 拖拽文件计算 MD5/SHA-256，纯本地 |
| 23 | 图片转 Base64 | 拖拽上传，输出 data URI，可预览 |

### 阶段二：格式化扩展（P1）

| # | 工具 | 说明 |
|---|---|---|
| 24 | XML 格式化 | 格式化 + 压缩 + 校验 |
| 25 | YAML ↔ JSON | 双向转换 + 格式化 |
| 26 | URL 解析器 | 拆解 URL 各字段，query 参数表格化 |
| 27 | 时间计算器 | 两日期之差、日期加减 |
| 28 | HTML/CSS/JS 压缩 | 纯前端压缩，去注释/空白 |

### 阶段三：文本增强（P1）
